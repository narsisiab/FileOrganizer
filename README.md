# File Organizer

A simple Python application that automatically organizes files into folders based on their file types.

## Features

- Organizes images, videos, documents, and audio files
- Simple Tkinter graphical interface
- Automatic folder creation
- Error handling for invalid paths

## Technologies Used

- Python
- Tkinter
- os
- shutil

## How to Run

1. Clone the repository
2. Run:

```bash
python gui.py
